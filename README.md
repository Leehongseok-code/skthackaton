# 자폐 아동 미술/음악 치료 서비스 유달리(U-Dalle)🎨

<br>

## 🖥️ 프로젝트 소개
`U-Dalle`는 자폐 아동을 위한 미술/음악 치료 서비스입니다. <br>
`ChatGPT api`와 `Dall-e api`를 사용하여 자폐 아동은 자신이 생각하는 콜라보를 만들어 볼 수 있습니다. <br><br>
<br><br>

## ⏰ 프로젝트 기간
2023.08.25 ~ 2023.09.03
